Building a configuration UI for a wearable device

Background

From 2015 to 2020, I designed and developed web applications for a medical-grade wearable device, using React, Redux and SVG.

Problem

Our system supported the placement of small biometric sensors on over 30 different body locations. However, there was potential for confusion between these locations (e.g. mistaking "Flexor Digitorum" for "Extensor Digitorum") and each location required a specific sensor orientation. Our customers needed an elegant, foolproof way to configure sensors.
